This page displays a user profile.

Currently the profile information may only be changed by the system administrator. Profile information may be viewed here. The ID is a sequential identifier number used internally by Stork. The Login is a friendly name identifying the user. It is unique for a given authentication method. The Email is an optional property. The Group is a label of a group that the user belongs to. The Authentication is an identifier of the authentication method used to login the specific user. The External ID is an identifier given by the external authentication method. It is not specified if the internal authentication method is used.
